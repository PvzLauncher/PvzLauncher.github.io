# 下载游戏

本章节指导您完成《植物大战僵尸》游戏及修改器的下载操作。

我们拥有一个丰富且持续更新的游戏库。截至本文编写时，游戏库已收录 **40+** 款可下载内容（包括**原版、改版和修改器**）。

所有资源均可通过启动器**免费、高速**下载，并在下载完成后自动解压，无需手动操作。

如果你没有在游戏库中找到想要的改版，欢迎通过 [Issue](https://github.com/PvzLauncher/PvzLauncher/issues/new?template=download.yml) 提交申请，我们会尽力将其添加到游戏库中。

![download-detail](/image/download-detail.png)

首先，确定好你要安装的游戏版本，并点击这个版本以进入详细信息页面

![download-info](/image/download-info.png)

点击右侧蓝色的“下载”按钮，将游戏版本加入任务列表中

![download-task](/image/download-task.png)

等待全自动下载安装完毕后，此版本就会作为可以启动的游戏版本被加入管理列表内，您就可以选中它并启动了，选中操作详见[游戏管理](/tutorial/Windows/manage.md)
